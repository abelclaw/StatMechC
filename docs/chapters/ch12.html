Chapter 12

Bose-Einstein Condensation

1. Introduction

Bose-Einstein condensationA quantum phase transition in which a macroscopic number of bosons simultaneously occupy the ground state. Unlike a classical gas cooling, where particles spread roughly evenly among low-energy states, BEC concentrates particles anomalously in a single state. is a quantum phenomenon in Bose gases in which a large number of bosons simultaneously occupy the ground state of a system. Bose-Einstein condensates were predicted in 1925 by Bose and discovered experimentally 70 years later by Wieman, Cornell and Ketterle who shared the Nobel prize for their discovery in 2001. More precisely, these scientists constructed an experiment where the phase transition to Bose-Einstein condensation could be clearly seen and measured. Many quantum phenomena such as superconductivity, superfluids or lasers can also be understood as Bose-Einstein condensation.

Your first thought might be, of course a lot of bosons are in the ground state! After all, there is no quantum effect preventing them from being in the ground state (no Pauli exclusion), and naturally particles want to be in the state of lowest energy. This is a good thought; let's follow through. How many bosons do you expect in the ground state? Well, say they obey Maxwell-Boltzmann statistics, so that $n_i \sim e^{-\epsilon_i/k_BT}$. This function is pretty flat for $\epsilon_i \ll k_BT$, so we would expect that if there are say 100 states below $k_BT$ then each one should have roughly the same number of particles in it — nothing too special about the ground state. Thus, if you want a sizable fraction, say 1/2 the particles, to be in the ground state, you would have to get $k_BT$ down below the energy of the first excited state $\epsilon_1$. This argument is correct for Maxwell-Boltzmann statistics, and we'll flesh it out more in a moment. The amazing thing is that with Bose-Einstein statistics the argument completely fails — you can find more than half of the particles in the ground state even for temperatures with $k_BT \gg \epsilon_1$.

Bose-Einstein condensation is tricky to explain, so we'll approach it different ways. First, we'll try to understand what it is about Bose-Einstein statistics that allows condensation to happen through a simple system that we can solve in the canonical ensemble. Then we'll do the general case using the grand-canonical ensemble, first numerically, and then through analytic expansions.

In this lecture, it will be helpful to set the ground state energy to zero: $\epsilon_0 = 0$. By setting it to zero, we mean that we list all energies as relative to the ground state. Shifting all the energies as well as the chemical potential in this way will have no effect on the physics and can be done without loss of generality.

2. Two-state System: Canonical Ensemble

Consider a system with $N$ particles but only two possible energy states: $\epsilon_0 = 0$ and $\epsilon_1 = \epsilon$. Because there are only two states, we can study this system in the canonical ensemble for both Maxwell-Boltzmann statistics and Bose-Einstein statistics. We'll see that even in this two-state system the ground state occupancy can be much larger than $\frac{N}{2}$ even when $k_BT \gg \epsilon$ with Bose-Einstein statistics. With the large number of states present in any realistic system, the canonical ensemble will not be tractable and we will have to resort to the more abstract grand canonical ensemble.

Let's start with the simplest case of our 2-state system, $N = 1$. If there is only one particle, then for any statistics, the partition function is

$$Z_1 = \sum_k e^{-\beta E_k} = 1 + e^{-\beta\epsilon}$$(1)

The probability of finding the particle in the ground state is $P_\text{ground} = \frac{1}{Z_1}e^{-\beta\epsilon_0} = \frac{1}{Z_1}$ and so the expected fraction of particles in the ground state is

$$\frac{\langle N_\text{ground}\rangle}{N} = 1 \cdot P_\text{ground} + 0 \cdot P_\text{not-ground} = \frac{1}{Z_1} = \frac{1}{1 + e^{-\beta\epsilon}}$$(2)

Again, this holds for any statistics, since there is only one particle.

Maxwell-Boltzmann case

Now say there are $N$ particles. With Maxwell-Boltzmann statistics, the probability of finding any particle in the ground state is independent of the probability of finding any other particle anywhere. This implies that the $N$ particle partition function is related to the 1 particle one by

$$Z_N^\text{MB} = \frac{1}{N!}(Z_1)^N = \frac{1}{N!}(1 + e^{-\beta\epsilon})^N$$(3)

Of the $2^N$ microstates, there is only one microstate with all the particles in the ground state, so $P_\text{all ground} = \frac{1}{Z_N^\text{MB}}\frac{1}{N!}$. To confirm that this is correct, let us verify that the probabilities sum to one. There are $N$ states with 1 particle in the excited state, $\binom{N}{2}$ states with 2 particles in the excited state and so on. So the sum of probabilities is

$$\sum_k P_k = \frac{1}{N! Z_N^\text{MB}}\left[1 + Ne^{-\beta\epsilon} + \binom{N}{2}e^{-2\beta\epsilon} + \cdots + \binom{N}{N}e^{-N\beta\epsilon}\right] = \frac{1}{N! Z_N^\text{MB}}(1 + e^{-\beta\epsilon})^N = 1$$(5)

To compute the expected number in the ground state, we multiply each term in this sum by the ground state occupancy:

$$\langle N_\text{ground}^\text{MB}\rangle = \frac{1}{N! Z_N^\text{MB}}\left(N \cdot 1 + (N-1) \cdot Ne^{-\beta\epsilon} + (N-2)\cdot\binom{N}{2}e^{-2\beta\epsilon} + \cdots + 0\cdot\binom{N}{N}e^{-N\beta\epsilon}\right)$$(6)
$$\langle N_\text{ground}^\text{MB}\rangle = \frac{N}{e^{-\beta\epsilon} + 1}$$(7)

You can check this sum in Mathematica. Note that at large $T$ ($\beta \to 0$), $\langle N_\text{ground}^\text{MB}\rangle$ goes to $\frac{N}{2}$: half the particles are in the ground state, half in the excited state.

Bose-Einstein case

With Bose-Einstein statistics there is only one state with $m$ particles in the ground state and $N - m$ particles in the excited state. So there are only $N + 1$ possible states all together and

$$Z_N^\text{BE} = 1 + e^{-\beta\epsilon} + e^{-2\beta\epsilon} + \cdots + e^{-N\beta\epsilon} = \frac{1 - e^{-(N+1)\beta\epsilon}}{1 - e^{-\beta\epsilon}}$$(8)

Then the expected number in the ground state is

$$\langle N_\text{ground}^\text{BE}\rangle = \frac{1}{Z_N^\text{BE}}\left[N \cdot 1 + (N-1)\cdot e^{-\beta\epsilon} + (N-2)\cdot e^{-2\beta\epsilon} + \cdots + 0 \cdot e^{-N\beta\epsilon}\right]$$(9)
$$\langle N_\text{ground}^\text{BE}\rangle = \frac{1}{e^{-\beta\epsilon} - 1} + \frac{N+1}{1 - e^{-(N+1)\beta\epsilon}}$$(10)

Let us look at Eqs. (7) and (10) numerically for $N = 100$:

What Makes BEC Special

To emphasize how strange Bose-Einstein condensation is, remember that at $k_BT \gg \epsilon$ we should be able to use $e^{-\beta\epsilon} \approx 1$ independent of $N$. That is, the ground state and first excited state should have pretty similar thermodynamic properties and occupancy at high temperature. This is not what we are finding. Instead, for $N = 10$ million, with a temperature 1 million times the excited state energy, 90% of the atoms are in the ground state and only 10% are in the excited state.

Bose-Einstein condensation is a phase transition whereby the ground state becomes highly occupied. What is the critical temperature for this to occur? There are only two dimensionless numbers we can work with: $\frac{k_BT}{\epsilon}$ and $N$. We want $N$ to be large, $N \gg 1$. Then we can expand in two limits $\frac{k_BT}{\epsilon} \gg N$ (i.e. $\beta\epsilon N \ll 1$) and $\frac{k_BT}{\epsilon} \ll N$ (i.e. $\beta\epsilon N \gg 1$). Expanding Eq. (10) in the first limit gives

$$\frac{\langle N_\text{ground}^\text{BE}\rangle}{N} = \frac{1}{2} + \frac{N\epsilon}{12k_BT} + \cdots \qquad \left(\frac{k_BT}{\epsilon} \gg N\right)$$(11)

This is the true high-temperature limit, where the classical behavior $\frac{\langle N_\text{ground}^\text{BE}\rangle}{N} \to \frac{\langle N_\text{ground}^\text{MB}\rangle}{N} \approx \frac{1}{2}$ is approached as $T \to \infty$. In the second limit $N \gg \frac{k_BT}{\epsilon} \gg 1$, the temperature is large, but not too large, and the expansion gives a different result

$$\frac{\langle N_\text{ground}^\text{BE}\rangle}{N} = 1 - \frac{k_BT}{N\epsilon} + \cdots \qquad \left(N \gg \frac{k_BT}{\epsilon}\right)$$(12)

This limit shows the growth of $\frac{\langle N_\text{ground}^\text{BE}\rangle}{N}$ toward 1 as $T \to 0$. The comparison of these approximations to Eq. (10) looks like:

The crossover point is roughly where the approximations used for our expansions break down. The first term is the same order as the second term in Eq. (11) when $k_BT = \frac{N\epsilon}{6}$. For Eq. Thus we find a critical temperature $T_c \sim \frac{N\epsilon}{6k_B} \sim \frac{N\epsilon}{k_B}$ for this two state model indicating the onset of Bose-Einstein condensation. (The crossover point in this 2-state example is not at a precise temperature, as you can see from the plot. $T_c$ will become precise when we consider a realistic system with a large number of states in the next section.)

Now, this 2-state model is not a realistic approximation to any physical system. It turns out to be very difficult to calculate $\langle N_\text{ground}^\text{BE}\rangle$ in the canonical ensemble for a realistic system that has an infinite number of states. The difficulty is that we have to count the number of ways of allocating $N$ particles to the states, and then to perform the sum over occupancies of the ground state times Boltzmann factors. It turns out to be much easier to compute the general case using the grand-canonical ensemble with $\mu$ instead of $N$, as we will now see.

3. Grand Canonical Ensemble

With Bose-Einstein statistics, we determined that using the grand canonical ensemble the expected number of particles in a state $i$ is

$$\langle N_i \rangle = \frac{1}{e^{\beta(\epsilon_i - \mu)} - 1}$$(13)

with $\epsilon_i$ the energy of the state $i$. So the expected number of particles in the ground state ($\epsilon_i = 0$) is

$$\langle N_\text{ground}\rangle = \frac{1}{e^{-\beta\mu} - 1}$$(14)

Interactive: Ground State Occupation vs Chemical Potential

Recall that for Bose gases, $\mu$ will always be lower than all the energies (i.e. $\mu < 0$ when $\epsilon_0 = 0$). We can see this explicitly from the plot since there is a singularity at $\mu = 0$. This singularity, that $N_\text{ground} \to \infty$ as $\mu \to 0$ is not physical. Recall that in the grand canonical ensemble we do not fix $N$, so the same distribution has to allow for arbitrarily large $N$. If we know $N$ then we have to trade $\mu$ for $N$ by imposing the constraint $\sum_i \langle N_i \rangle = N$. This is not so easy, but we can do it.

To replace $\mu$ by $N$, we first invert Eq. (14) to solve for $\mu$ in terms of $\langle N_\text{ground}\rangle$:

$$e^{-\beta\mu} = \frac{1}{\langle N_\text{ground}\rangle} + 1$$(15)

Our strategy will then be to find the ground state occupancy by using the constraint that the total number of particles is $N$. That is, we will compute

$$N = \sum_{i=0}^{\infty}\langle N_i \rangle = \sum_{i=0}^{\infty}\frac{1}{e^{\beta\epsilon_i}\!\left(\frac{1}{\langle N_\text{ground}\rangle} + 1\right) - 1}$$(16)

Once we work out the $\epsilon_i$ we can do the sum numerically or analytically and therefore find $\frac{\langle N_\text{ground}\rangle}{N}$.

Before beginning the calculation, let us quickly ask about the non-ground state occupancies. Since

$$\mu = -\frac{1}{\beta}\ln\!\left[\frac{1}{\langle N_\text{ground}\rangle} + 1\right] < -\frac{1}{\beta}\ln\!\left[\frac{1}{N} + 1\right] < 0$$(17)

$\mu$ is always negative and gets closest to zero when $\langle N_\text{ground}\rangle$ is largest. Conversely, $\langle N_\text{ground}\rangle$ is largest when $\mu$ is closest to zero as can be seen in Eq. (14).

Why Only the Ground State Condenses

Can excited states have a large number of particles in them? The explosion of particles in the ground state arose because if $\mu \to 0$ then $e^{-\beta\mu} \to 1$ and $\langle N_\text{ground}\rangle = \frac{1}{e^{-\beta\mu} - 1} \to \infty$. For the first excited state:

$$\langle N_1 \rangle = \frac{1}{e^{\beta(\epsilon_1 - \mu)} - 1} = \frac{1}{e^{\beta\epsilon_1}\!\left(\frac{1}{\langle N_\text{ground}\rangle} + 1\right) - 1}$$

Since $\frac{1}{\langle N_\text{ground}\rangle} + 1 > 1$ and $e^{\beta\epsilon_1} > 1$, this can never get too large. Since $\mu < 0$ there is always a gap between $\mu$ and any energy other than the ground state, so $e^{\beta(\epsilon_1 - \mu)} > e^{\beta\epsilon_1} > 1$. Since $\mu$ cannot get arbitrarily close to $\epsilon_i$, condensation cannot happen in any excited state. The ground state is special.

3.1 Exact Numerical Solution

Now let us calculate $N$, and hence $\frac{\langle N_\text{ground}\rangle}{N}$. Bose-Einstein condensation is relevant at low temperature, where particles are non-relativistic. So consider a non-relativistic gas of monatomic bosonic atoms in a 3D box of size $L$. The allowed wavevectors of the system are $\vec{k}_n = \frac{\pi}{L}\vec{n}$ just like for photons or phonons, and the momenta are $\vec{p}_n = \hbar\vec{k}_n$ as always. In a non-relativistic system, the energies are

$$\epsilon_n = \frac{\vec{p}_n^{\,2}}{2m} = \frac{\hbar^2\pi^2}{2mL^2}\vec{n}^{\,2}$$(19)

We have set the ground state to $\epsilon_0 = 0$ (rather than $\epsilon_0 = mc^2$), since the absolute energy scale will be irrelevant. Another useful number is the gap to the first excited state

$$\epsilon_1 = \frac{\hbar^2\pi^2}{2mL^2}(1, 0, 0)^2 = \frac{\hbar^2\pi^2}{2mL^2}$$(20)

So that $\epsilon_n = \epsilon_1 n^2$. Then, from Eq. (16) we get

$$N = \sum_{n_x, n_y, n_z = 0}^{\infty}\frac{1}{e^{\beta\epsilon_1(n_x^2 + n_y^2 + n_z^2)}\!\left(\frac{1}{\langle N_\text{ground}\rangle} + 1\right) - 1}$$(22)

This formula lets us compute $N$ given $\langle N_\text{ground}\rangle$ and $\beta\epsilon_1$. For example, if $\langle N_\text{ground}\rangle = 80$ and $T = 10\frac{\epsilon_1}{k_B}$ then doing the sum numerically gives $N = 167.5$. This means with 167.5 particles at $T = 10\frac{\epsilon_1}{k_B}$, then 80 will be in the ground state.

What we really want is to specify $N$ and $T$ and find $\langle N_\text{ground}\rangle$. To get this function, we need to do the sum in Eq. (22) and then solve for $\langle N_\text{ground}\rangle$ in terms of $\beta\epsilon_1$ and $N$. Unfortunately, we cannot do the sum exactly, but at least we can do it numerically. For $N = 100$ we find the numerical solution for $\frac{\langle N_\text{ground}\rangle}{N}$ has the form:

3.2 Approximate Analytical Solution

Having determined the exact solution numerically, let us proceed to use an analytical approach to determine some scaling relations and the transition temperature.

▶ Derivation: BEC Critical Temperature and Condensate Fraction
Step 1

As with the phonon or photon gas, we first transform the sum to an integral via

$$\sum_{\vec{n}} \to \frac{1}{8}\int_0^{\infty} 4\pi n^2\,dn$$

where the $\frac{1}{8}$ comes from $\vec{n}$ being a vector of whole numbers. We want to convert $n$ to $\epsilon$, using $\epsilon = \epsilon_1 n^2$ so $d\epsilon = 2\epsilon_1 n\,dn$:

$$\sum_{\vec{n}} \to \frac{\pi}{2}\int_0^{\infty} n\,n\,dn = \frac{\pi}{2}\int_0^{\infty}\sqrt{\frac{\epsilon}{\epsilon_1}}\frac{d\epsilon}{2\epsilon_1} = \frac{\pi}{4\epsilon_1^{3/2}}\int_0^{\infty}\sqrt{\epsilon}\,d\epsilon$$

This defines the density of states:

$$g(\epsilon) = \frac{\pi}{4\epsilon_1^{3/2}}\sqrt{\epsilon}$$
Step 2

We would like to integrate to find $N$:

$$N = \int_0^{\infty} g(\epsilon)\langle n_\epsilon\rangle\,d\epsilon = \frac{\pi}{4\epsilon_1^{3/2}}\int_0^{\infty}d\epsilon\frac{\sqrt{\epsilon}}{e^{\beta(\epsilon - \mu)} - 1}$$

However, the ground state is qualitatively different. The density of states $g(\epsilon) \propto \sqrt{\epsilon}$ vanishes at $\epsilon = 0$, so the integral misses the ground state entirely! We must separate $N$ into ground state and excited state contributions: $N = \langle N_\text{ground}\rangle + \langle N_\text{excited}\rangle$.

Step 3

For the excited states, since $e^{-\beta\mu} \approx 1$ when $\langle N_\text{ground}\rangle \gtrsim 1$ (the BEC regime), we can set $\mu = 0$:

$$\langle N_\text{excited}\rangle \approx \frac{\pi}{4\epsilon_1^{3/2}}\int_{\epsilon_1}^{\infty}d\epsilon\frac{\sqrt{\epsilon}}{e^{\beta\epsilon} - 1} = \left(\frac{\pi k_BT}{4\epsilon_1}\right)^{3/2}\zeta_{3/2}\left[1 + O\!\left(\sqrt{\frac{\epsilon_1}{k_BT}}\right) + \cdots\right]$$

where $\zeta_{3/2} = \zeta\!\left(\frac{3}{2}\right) \approx 2.612$.

Step 4

The $\mu = 0$ approximation breaks down when $\langle N_\text{ground}\rangle \approx 0$ and $\langle N_\text{excited}\rangle \approx N$. Setting $\langle N_\text{excited}\rangle = N$ gives the critical temperature:

$$N = \zeta_{3/2}\!\left(\frac{\pi k_BT_c}{4\epsilon_1}\right)^{3/2} = 2.612\!\left(\frac{mk_BT_c}{8\hbar^2\pi}\right)^{3/2}V$$

Solving for $T_c$:

$$T_c = \frac{4\epsilon_1}{\pi k_B}\!\left(\frac{N}{\zeta_{3/2}}\right)^{2/3} = 3.31\,\frac{\hbar^2}{k_Bm}\!\left(\frac{N}{V}\right)^{2/3}$$

The density of states for a non-relativistic Bose gas in a box is

$$g(\epsilon) = \frac{\pi}{4\epsilon_1^{3/2}}\sqrt{\epsilon}$$(28)

The critical temperature is

$$T_c = \frac{4\epsilon_1}{\pi k_B}\!\left(\frac{N}{\zeta_{3/2}}\right)^{2/3} = 3.31\,\frac{\hbar^2}{k_B m}\!\left(\frac{N}{V}\right)^{2/3}$$(33)

This lets us write the fraction of excited particles as

$$\frac{N_\text{excited}}{N} = \left(\frac{T}{T_c}\right)^{3/2}, \qquad T < T_c$$(34)

Thus, the fraction in the ground state is

$$\frac{\langle N_\text{ground}\rangle}{N} = \frac{N - \langle N_\text{excited}\rangle}{N} = 1 - \left(\frac{T}{T_c}\right)^{3/2}$$(35)

4. Experimental Evidence

Even though the transition temperature to the BEC is well above the first excited state energy, one still needs to get a system of bosons very very cold to produce a BEC. It took 70 years from when BECs were first conjectured theoretically (by Satyendra Nath Bose in 1924) to when technology to cool atoms was sufficiently advanced that the BEC could be produced and detected (Cornell, Wieman and Ketterle in 1995). The tricky part is that normally when you cool a gas of atoms, they solidify. A solid is not a BEC. So you need to cool the gas while keeping the density low. However, since $T_c = 3.31\frac{\hbar^2}{k_B m}\!\left(\frac{N}{V}\right)^{2/3}$ if you keep the density low you need very very low (ultracold) temperatures.

Since $T_c = 3.31\frac{\hbar^2}{k_Bm}\!\left(\frac{N}{V}\right)^{2/3}$ depends inversely on the mass of the atoms, lighter atoms allow the critical temperature to be higher. You might therefore think hydrogen is the easiest element to cool to see a BEC form. Unfortunately, in the mid 1990s, lasers weren't available that could operate at frequencies conducive to cooling hydrogen. Cornell and Wieman, and Ketterle, used Rubidium atoms$^{87}$Rb: an alkali metal atom with 37 protons, 50 neutrons, and 37 electrons (87 fermions total, making the atom a boson). Its convenient energy level structure made it ideal for laser cooling in the 1990s., $^{87}$Rb to form the BEC. Rubidium has a convenient set of energy levels that were well suited to the available laser cooling technologies at the time. They were able to cool around 2000 atoms using magneto-optical trapsMOTs use a combination of laser beams and magnetic fields to slow down and trap atoms. Combined with evaporative cooling (removing the most energetic atoms), temperatures of nanokelvin can be achieved. (MOTs) along with laser and evaporative cooling techniques to the nanokelvin temperature scales required for the BEC. They found that below $T_c \approx 170\,\text{nK}$ Bose-Einstein condensation can be seen. This critical temperature is in excellent agreement with the general formula $T_c = 3.31\frac{\hbar^2}{k_Bm}\!\left(\frac{N}{V}\right)^{2/3}$ when you plug in the density they were able to achieve.

In order to see the BEC, one needs to be able to distinguish atoms in the ground state from atoms in the excited states. The basic feature that makes this possible is that the ground state atoms all have smaller wavenumbers and hence slower velocities than the other states. So if you remove the magnetic trap, the atoms will start to spread, with the ground state atoms spreading more slowly. Thus the atoms' positions after a short time indicate their initial velocities. The scientists photographed (i.e. illuminated the system with a resonant laser pulse) the system after $t = 100\,\text{ms}$ and found a high density of atoms that had not moved very far:

In this figure, from Wieman and Cornell's group at the JILA laboratory in Boulder, we see the distribution of atoms in the BEC at different temperatures. The critical temperature is $T_c \approx 170\,\text{nK}$. Above this temperature (left), the distribution is pretty smooth, consistent with a Maxwell-Boltzmann velocity distribution. Below the critical temperature you can clearly see the higher density corresponding to an anomalously large occupancy of the ground state, consistent with expectations from a BEC. The right image shows the BEC at even lower temperature, where the occupancy is even higher. Direct observation of thermodynamic properties of the system, such as the energy density and heat capacity (particularly by Ketterle's group at MIT), further confirmed that this system was a BEC.

Since their discovery BECs have continued to exhibit some amazing and unusual properties. Because of their coherence (all the atoms are in the same state), they can manifest quantum phenomena at larger scales than electrons and are in many ways more controllable than electrons. A number of groups at Harvard and MIT study BECs.

BEC Research at Harvard

Prof. Greiner (Harvard) uses lasers to localize rubidium atoms in an optical lattice. By controlling the spacing and lattice properties, he can fine tune the system, essentially choosing whatever Hamiltonian he wants. Once the rubidium atoms form a BEC, they exhibit strongly correlated behavior that can be connected to the properties of the Hamiltonian. Such an approach may lead the way to building quantum computers with longer coherence times, or to understanding what material properties might be most likely to produce room-temperature superconductors.

Prof. Hau (Harvard) constructed a BEC of sodium atoms in 1999. Normally, a laser tuned to a hyperfine splitting of the sodium levels would be absorbed and so sodium appears opaque to this frequency. However, Hau was able to entangle the ground and excited states of sodium using photons of a different laser in such a way that she could adjust the transparency of sodium to the first laser. The result is that she could manipulate the dispersion relation of light propagating through the sodium BEC and achieve arbitrarily small group velocities. In her first paper on the subject, she slowed light down to $17\,\text{m/s}$ with this technique. Subsequently, she was able to get light to stop completely.

Superfluids and Superconductors

The BECs produced in laboratories have carefully controlled properties, and very restricted interactions. A BEC of rubidium or sodium is unstable to condense in solid form and must be held apart with some careful tricks using magnetic fields and optical traps. Helium is a noble gas that naturally has very weak interactions and will only solidify at very high pressure. At low temperature and pressure, it forms a Bose-Einstein condensate called a superfluidA fluid with zero viscosity. Liquid $^4$He becomes a superfluid below 2.17 K (the lambda point). The superfluid can flow without friction, climb container walls, and exhibits quantized vortices.. Liquid $^4$He is a BEC of helium atoms. Although $^3$He is fermionic, pairs of $^3$He atoms are bosonic, so liquid $^3$He can be thought of as forming due to the pairing of helium atoms. Superfluids have zero viscosity. This is closely related to the bosons all being in the same state, but in this case, the state is not the zero-momentum state but one of non-zero momentum since there is a density current flowing through the fluid.

Bose-Einstein condensation is also related to superconductorsMaterials with exactly zero electrical resistance below a critical temperature. In BCS theory, electrons form Cooper pairs (bosons) that condense, allowing current to flow without resistance. The Meissner effect expels magnetic fields from the superconductor.. In a superconductor, like solid mercury at $T < 4.2\,\text{K}$, there is no resistivity. In the BCS theoryBardeen-Cooper-Schrieffer theory (1957): explains superconductivity through the condensation of Cooper pairs — pairs of electrons bound by phonon-mediated interactions. The pairs are bosonic and undergo BEC., the superconductivity is explained through the condensation of pairs of electrons called Cooper pairs. These pairs act like bosons and form a condensate at low temperature. Because electrons are charged, so are the Cooper pairs. The condensation of charged pairs screens the magnetic field in the superconductor, allowing the charged current to flow with zero resistance. Thus superconductors are much like superfluids with charged bosons instead of neutral ones.

5. Summary

Bose-Einstein condensation is a phenomenon whereby systems of bosons tend towards configurations where most of the bosons are in the identical state. Typically, you might think that at a temperature $T$, all the states with energy $\epsilon \lesssim k_BT$ would be occupied. Instead, what happens is that a single state, which can have energy $\epsilon_0 \ll k_BT$ gets an order-one fraction of the particles, while other states, even ones with $\epsilon \ll k_BT$ as well, are hardly occupied at all.

There are various ways to understand Bose-Einstein condensation. We looked at a toy model using the canonical ensemble where we saw the condensation happen. A more powerful tool is the grand canonical ensemble. In the grand canonical ensemble, the chemical potential $\mu$ is key. The condensation happens for the state with $\epsilon_0 \approx \mu$ for which the expected occupation $\langle n_i \rangle = \frac{1}{e^{\beta(\epsilon_i - \mu)} - 1}$ blows up.

We found that there is a phase transition. Above a critical temperature $T_c = \frac{4\epsilon_1}{\pi k_B}\!\left(\frac{N}{\zeta_{3/2}}\right)^{2/3}$, all states with $\epsilon \lesssim k_BT$ are evenly occupied. However, below $T_c$, the occupation number of the ground state grows like $\frac{\langle N_\text{ground}\rangle}{N} = 1 - \left(\frac{T}{T_c}\right)^{3/2}$.

Bose-Einstein condensates include superfluids, superconductors and lasers. They were predicted in 1924 but not seen in the lab until 1995.

Chapter Summary

  • Bose-Einstein condensation is a phase transition where a macroscopic fraction of bosons occupy the ground state, even at temperatures $k_BT \gg \epsilon_1$.
  • In a two-state toy model, the condensation can be seen in the canonical ensemble: BE statistics gives much higher ground state occupancy than MB statistics at the same temperature.
  • In the grand canonical ensemble, the chemical potential $\mu$ approaches the ground state energy as $T$ decreases, causing $\langle N_\text{ground}\rangle = \frac{1}{e^{-\beta\mu} - 1}$ to diverge.
  • The critical temperature is $T_c = 3.31\frac{\hbar^2}{k_Bm}\!\left(\frac{N}{V}\right)^{2/3}$ — it scales as $N^{2/3}$, well above the first excited state energy.
  • Below $T_c$, the condensate fraction grows as $\frac{N_0}{N} = 1 - \left(\frac{T}{T_c}\right)^{3/2}$.
  • BEC was first observed in 1995 in ultracold $^{87}$Rb at $T_c \approx 170\,\text{nK}$ (Nobel Prize 2001).
  • Related phenomena include superfluidity in liquid $^4$He and superconductivity via Cooper pair condensation.
